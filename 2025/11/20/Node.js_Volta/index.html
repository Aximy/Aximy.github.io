介绍

详细内容参考：官方文档

Volta 的任务是管理你的 JavaScript 命令行工具，例如 node 、 npm 、 yarn ，或作为 JavaScript 包一部分分发的可执行文件。

特点：

  • 速度快
  • 跨平台支持
  • 可以无缝切换版本
